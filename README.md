# Snake

### Description
A simple snake game made with ES6.

### Build Steps
```
yarn
```

```
yarn run build
```

### To Run

This package is prebuild into the dist folder. Open dist/index.html to see the game in action!

Optionally you can run:

```
webpack-dev-server
```

Then open http://localhost:9000
